# 남은 시간 계산기
지정한 날짜와 시간까지 남은 시간을 계산해주는 간단한 GUI 애플리케이션입니다. Python과 Tkinter로 만들어졌으며, 사용자가 미래의 날짜와 시간을 입력하면 실시간으로 카운트다운을 표시합니다.

## Requirements
- Python 3.13 에서 만듦.
- Tkinter

## Install & Run
가볍게 만든 프로그램이라 특별한 실행파일은 만들지 않았습니다. 클론해서 쓰세요.
